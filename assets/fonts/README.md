# Fonts

Tracked font assets that are packaged from this repository live here.

Any `.ttf`, `.ttc`, or `.otf` file placed here is packaged automatically.

Current font assets:

- `Nova.ttf`
- `Nova-Italic.ttf`
- `jetbrains-mono-nerd/JetBrainsMonoNerdFontMono-Regular.ttf`
- `jetbrains-mono-nerd/JetBrainsMonoNerdFontMono-Bold.ttf`
- `jetbrains-mono-nerd/JetBrainsMonoNerdFontMono-Italic.ttf`
- `jetbrains-mono-nerd/JetBrainsMonoNerdFontMono-BoldItalic.ttf`

`Nova.ttf` is the default Nova Regular face used across the workstation, while `Nova-Italic.ttf` is packaged alongside it.

The JetBrains Mono Nerd Font Mono variants are tracked here so terminal prompts and UI fallbacks can render Nerd Font glyphs such as folder and Git icons correctly.
